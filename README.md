# CPU-Simulator
This project presents a desktop-based CPU Scheduling Simulator developed  using Python.  The simulator demonstrates classical CPU scheduling algorithms, including  FCFS, SJF, SRTF, Priority, and Round Robin. 
It allows users to input process details such as Process ID, Burst Time, Arrival 
Time, Priority, and Time Quantum. 
The system computes scheduling metrics automatically, including Waiting 
Time, Turnaround Time, Average Waiting Time, and Total CPU Time. 
The simulator also provides graphical visualization of process execution using 
Gantt charts. 
The interface is user-friendly, with a pastel theme, centered window, and 
enlarged fonts for clarity. 
Processes are displayed in a tabular format with options to add, edit, and 
delete entries. 
The system supports both preemptive and non-preemptive scheduling 
algorithms. 
It serves as an educational tool for understanding process scheduling and CPU 
allocation in operating systems. 
The simulator enhances conceptual understanding by allowing students to 
observe algorithm behaviour interactively. 
It demonstrates the effect of different scheduling policies on performance 
metrics. 
The project emphasizes Process Management in operating systems and 
highlights real-time computation of scheduling outcomes. 
It can be used for academic laboratory experiments, assignments, and project 
demonstrations. 
The design is modular and scalable, allowing easy addition of new algorithms 
or features in the future. 
Overall, the simulator provides a practical, interactive learning platform for 
operating system students.
